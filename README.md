# ☕ Cafe Sales Analysis

## Project Overview
This project analyzes cafe sales data to uncover insights on customer behavior, product performance, and sales trends.

---

## Tools Used
- Python (Pandas, Seaborn, Matplotlib)
- Power BI

---

## Key Insights
- Takeaway orders dominate over in-store purchases
- Coffee and Cake are top-selling items
- Missing location data identified and handled

---

## Dashboard Features
- Total Sales KPI
- Quantity Sold KPI
- Sales by Category
- Sales by Location

---

## 📂 Files Included
- `Cafe_Sales_Project.ipynb` → Data analysis
- `Cafe_Sales_Dashboard.pbix` → Power BI dashboard
- `dashboard.png` → Dashboard preview
